RECOMMENDED VISUAL ASSETS FOR CONFERENCE PRESENTATION
Spatial Mapping of Energy Poverty in Bangladesh — MEPI Study
=============================================================

Preparation Note
----------------
All charts and maps listed below are generated by the project's Python scripts.
Run the following commands to produce the assets before the presentation:

  python generate_all_visualizations.py     # charts and graphs
  python generate_all_maps_organized.py     # spatial maps
  python generate_spatial_maps_only.py      # static spatial PNGs

Output directories:
  visualizations/         — charts and graphs (already present)
  map_outputs/            — organised map outputs
  ~/spatial_maps_png/     — standalone spatial PNG maps

Resolution guidelines:
  Print / poster:  300 DPI, PNG, minimum 3000 × 2000 px
  Screen / slides: 96 DPI,  PNG, minimum 1920 × 1080 px
  Interactive:     Folium HTML (no DPI constraint)
