# Bangladesh Shapefiles Directory

Place Bangladesh upazila boundary shapefiles here.

## Required Files

For a Shapefile format, you need all four files:

```
shapefiles/
├── bgd_adm2.shp    ← geometry
├── bgd_adm2.dbf    ← attribute table
├── bgd_adm2.shx    ← index
└── bgd_adm2.prj    ← projection info
```

For GeoJSON format, a single file is sufficient:

```
shapefiles/
└── bgd_adm2.geojson
```
